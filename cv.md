### __Code example:__
Task from CodeWars: _Complete the solution so that it reverses the string passed into it._

Solution:

```
function solution(str){
  let reverseWord = ''
  for (let i = str.length - 1; i >= 0; i--) {
    reverseWord += str[i]
  }
  return reverseWord
}
```
